Cursos disponibles

Angular

Precio: Gratis

Cupos: 5

En este curso de Angular, aprenderás a desarrollar aplicaciones web de una sola página usando el popular framework de Google. Ideal para quienes desean construir interfaces de usuario modernas y eficientes.

React

Precio: Gratis

Cupos: 8

Este curso de React te enseñará a construir interfaces de usuario reutilizables y componentizadas. Aprenderás los conceptos fundamentales, como JSX, props, y el manejo del estado, usando la librería más popular de JavaScript desarrollada por Facebook.

Node.JS

Precio: Gratis

Cupos: 4

Aprende Node.js para crear aplicaciones del lado del servidor utilizando JavaScript. En este curso explorarás cómo construir una API REST, trabajar con bases de datos y manejar peticiones HTTP de manera eficiente.
